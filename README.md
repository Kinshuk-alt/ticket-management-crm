# Ticket Management CRM System

## Overview
A CRM based Ticket Management System developed using:

- Java
- Spring Boot
- MySQL
- Spring Security
- HTML
- CSS
- JavaScript

## Modules

### User Module
- Register user
- Login user
- Role management
- BCrypt password encryption

### Ticket Module
- Create ticket
- Assign ticket
- Update status
- Search/filter

### Comment Module
- Add comments
- View comments

### Dashboard
- Total users
- Total tickets
- Resolved tickets
- Reports

### Roles
ADMIN
AGENT
USER

## Technologies

Backend:
- Spring Boot
- JPA
- Hibernate
- MySQL

Frontend:
- HTML
- CSS
- JavaScript
